Deployment Vars
---------------

Secrets
*******

None Required

Host variables
**************

``ohmyzsh_theme``:
    * Required
    * Some themes requires manual installation

``ohmyzsh_plugins``:
    * Optional
    * Some plugins requires manual installation

``ohmyzsh_users``:
    * Optional
    * If not set, default to additional users in the group_vars/all.yml file


Group variables
***************

None Required
